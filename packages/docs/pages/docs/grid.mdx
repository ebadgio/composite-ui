# Grid

Grid is a div component built using Styled System's `grid`, `space`, and `layout` style functions.
It allows you to css flexbox props to declaratively build grid display ui layouts.

## Usage

```js
import { Grid } from 'composite-ui'; 
```

## Examples

```jsx live=true
<Grid gridTemplateColumns="repeat(3, 1fr)" gridGap={3}>
  <Box width="100%" height="35px" bg="black.0" />
  <Box width="100%" height="35px" bg="black.0" />
  <Box width="100%" height="35px" bg="black.0" />
</Grid>
```
