# Textarea

Textareas are for multi-line user text input. The `Textarea` component supports a variant design pattern whereby you can
draft different `Textarea` variants entirely from your theme object. 

Textareas also implement the `color`, `border`, and `space` style functions.


## Usage

```js
import { Textarea } from 'composite-ui';
```

## Variants

```jsx live=true
<Textarea 
  variant="outline" 
  width="100%" 
  placeholder="This is a textarea..."
/>
```
