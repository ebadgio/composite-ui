# Heading

Heading is a primitive html heading component that is capable of rendering `h1` through `h6`. It uses
the `typography`, `color`, and `margin` style functions.

## Usage

```js
import { Heading } from 'composite-ui';
```

## Examples

```jsx live=true
<Box>
  <Heading>H1</Heading>
  <Heading level={2}>H2</Heading>
  <Heading level={3}>H3</Heading>
  <Heading level={4}>H4</Heading>
  <Heading level={5}>H5</Heading>
  <Heading level={6}>H6</Heading>
</Box>
```
