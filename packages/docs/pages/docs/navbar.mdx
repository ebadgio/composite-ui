# Navbar

Navigation bars are used to create a persistent navigation experience across different pages of your application. The `Navbar` component
uses `borderBottom`, `boxShadow`, `color`, `height`, and `zIndex` style functions for flexible, theme based styling.

## Usage

```js
import { Navbar } from 'composite-ui';
```

## Examples


### Simple Navbar

```jsx live=true
function Layout() {
  
  return (
    <Box position="relative" width="100%">
      <Navbar
        position="static" // (just for demo)
        backgroundColor="#fff"
        title="App"
        boxShadow={1}
      />
    </Box>
  )
}
```

### Navbar with Tabs

```jsx live=true
function Layout() {
  const [activeTab, setActiveTab] = React.useState(0);
  const tabs = [{text: 'Home'}, {text: 'About'}]

  return (
    <Box position="relative" width="100%">
      <Navbar
        position="static"
        backgroundColor="#fff"
        title="App"
        boxShadow={1}>
      <Tabs 
        onTabChange={(_, id) => {setActiveTab(id)}} 
        variant="basic"
        activeTab={activeTab}
        tabs={tabs}
      />
      </Navbar>
    </Box>
  )
}
```
