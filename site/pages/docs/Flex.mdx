# Flex

Flex is a `div` component built using Styled System's `flexbox`, `space`, and `layout` 
style functions. It allows you to css flexbox props to declaratively build flexbox ui layouts. 

The accepted flexbox props are `alignItems`,`alignContent`, `justifyItems`, `justifyContent`, `flexWrap`, `flexDirection`, `flex`, `flexGrow`, 
`flexShrink`, `flexBasis`, `justifySelf`, `alignSelf`, `order`

## Usage

```js
import { Flex } from 'composite-ui'
```

## Examples

```jsx live=true
<Flex flexDirection="column" width="100%">
  <Flex width="100%" alignItems="center">
    <Box width="25%" height="40px" m={1} bg="black.1"/>
    <Box width="25%" height="40px" m={1} bg="black.1"/>
    <Box width="25%" height="40px" m={1} bg="black.1"/>
  </Flex>
  <Flex width="100%" alignItems="center" justifyContent="center">
    <Box width="25%" height="40px" m={1} bg="black.1"/>
    <Box width="25%" height="40px" m={1} bg="black.1"/>
    <Box width="25%" height="40px" m={1} bg="black.1"/>
  </Flex>
  <Flex width="100%" alignItems="center" justifyContent="flex-end">
    <Box width="25%" height="40px" m={1} bg="black.1"/>
    <Box width="25%" height="40px" m={1} bg="black.1"/>
    <Box width="25%" height="40px" m={1} bg="black.1"/>
  </Flex>
</Flex>
```
