# Box

The Box component is an abstract component that can be used to easily build ui layouts or as a primitive for building more complex and specific ui elements.

## Usage

```js
import { Box } from 'composite-ui';
```

## Examples

```jsx live=true
<Box 
  bg="black.1" 
  w="100%" 
  p={3} 
  color="white" 
  borderRadius={1}>
  This is a Box
</Box>
```

```jsx live=true
<Box 
  width="100%" 
  boxShadow={1} 
  p={3} 
  mb={3} 
  borderRadius={1}>
  This Box has shadow
</Box>
```
