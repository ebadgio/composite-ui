# Input

Inputs are for user text input. The `Input` component supports a variant design pattern whereby you can
draft different input variants entirely from your theme object. 

Inputs also implement the `color`, `border`, and `space` style functions.

## Usage

```js
import { Input } from 'composite-ui';
```

## Variants

```jsx live=true
<Input 
  variant="outline" 
  width="100%" 
  placeholder="This is an input..." 
/>
```
